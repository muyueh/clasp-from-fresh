## [Link](https://developers.google.com/apps-script/reference/slides/link)

### Methods

|                                                  Method                                                  |                                        Return type                                         |                                                                                                                                                                                                    Brief description                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getLinkType()](https://developers.google.com/apps-script/reference/slides/link#getLinkType())           | [LinkType](https://developers.google.com/apps-script/reference/slides/link-type)           | Returns the [LinkType](https://developers.google.com/apps-script/reference/slides/link-type).                                                                                                                                                                                                                                                                                                                           |
| [getLinkedSlide()](https://developers.google.com/apps-script/reference/slides/link#getLinkedSlide())     | [Slide](https://developers.google.com/apps-script/reference/slides/slide)                  | Returns the linked [Slide](https://developers.google.com/apps-script/reference/slides/slide) for non-URL links types, if it exists.                                                                                                                                                                                                                                                                                     |
| [getSlideId()](https://developers.google.com/apps-script/reference/slides/link#getSlideId())             | `String`                                                                                   | Returns the ID of the linked [Slide](https://developers.google.com/apps-script/reference/slides/slide) or `null` if the [LinkType](https://developers.google.com/apps-script/reference/slides/link-type) is not [LinkType.SLIDE_ID](https://developers.google.com/apps-script/reference/slides/link-type#SLIDE_ID).                                                                                                     |
| [getSlideIndex()](https://developers.google.com/apps-script/reference/slides/link#getSlideIndex())       | `Integer`                                                                                  | Returns the zero-based index of the linked [Slide](https://developers.google.com/apps-script/reference/slides/slide) or `null` if the [LinkType](https://developers.google.com/apps-script/reference/slides/link-type) is not [LinkType.SLIDE_INDEX](https://developers.google.com/apps-script/reference/slides/link-type#SLIDE_INDEX).                                                                                 |
| [getSlidePosition()](https://developers.google.com/apps-script/reference/slides/link#getSlidePosition()) | [SlidePosition](https://developers.google.com/apps-script/reference/slides/slide-position) | Returns the [SlidePosition](https://developers.google.com/apps-script/reference/slides/slide-position) of the linked [Slide](https://developers.google.com/apps-script/reference/slides/slide) or `null` if the [LinkType](https://developers.google.com/apps-script/reference/slides/link-type) is not [LinkType.SLIDE_POSITION](https://developers.google.com/apps-script/reference/slides/link-type#SLIDE_POSITION). |
| [getUrl()](https://developers.google.com/apps-script/reference/slides/link#getUrl())                     | `String`                                                                                   | Returns the URL to the external web page or `null` if the [LinkType](https://developers.google.com/apps-script/reference/slides/link-type) is not [LinkType.URL](https://developers.google.com/apps-script/reference/slides/link-type#URL).                                                                                                                                                                             |
