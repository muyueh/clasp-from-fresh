### Properties

|   Property    |  Type  |                Description                 |
|---------------|--------|--------------------------------------------|
| `UNSUPPORTED` | `Enum` | A slide linking mode that isn't supported. |
| `LINKED`      | `Enum` | Indicates that slides are linked.          |
| `NOT_LINKED`  | `Enum` | Indicates that slides aren't linked.       |
