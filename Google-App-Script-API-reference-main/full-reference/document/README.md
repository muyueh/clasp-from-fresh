# Document Service

Document

This service allows scripts to create, access, and modify Google Docs files.  

    // Open a document by ID.
    var doc = DocumentApp.openById('DOCUMENT_ID');

    // Create and open a document.
    doc = DocumentApp.create('Document Name');
