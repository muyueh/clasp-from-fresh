## [User](https://developers.google.com/apps-script/reference/drive/user)

### Methods

|                                            Method                                             | Return type |                    Brief description                     |
|-----------------------------------------------------------------------------------------------|-------------|----------------------------------------------------------|
| [getDomain()](https://developers.google.com/apps-script/reference/drive/user#getDomain())     | `String`    | Gets the domain name associated with the user's account. |
| [getEmail()](https://developers.google.com/apps-script/reference/drive/user#getEmail())       | `String`    | Gets the user's email address.                           |
| [getName()](https://developers.google.com/apps-script/reference/drive/user#getName())         | `String`    | Gets the user's name.                                    |
| [getPhotoUrl()](https://developers.google.com/apps-script/reference/drive/user#getPhotoUrl()) | `String`    | Gets the URL for the user's photo.                       |
