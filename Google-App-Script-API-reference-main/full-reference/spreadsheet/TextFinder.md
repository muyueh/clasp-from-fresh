## [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder)

### Methods

|                                                                   Method                                                                    |                                        Return type                                        |                                                                        Brief description                                                                         |
|---------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [findAll()](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#findAll())                                          | [Range[]](https://developers.google.com/apps-script/reference/spreadsheet/range)          | Returns all cells matching the search criteria.                                                                                                                  |
| [findNext()](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#findNext())                                        | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)            | Returns the next cell matching the search criteria.                                                                                                              |
| [findPrevious()](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#findPrevious())                                | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)            | Returns the previous cell matching the search criteria.                                                                                                          |
| [getCurrentMatch()](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#getCurrentMatch())                          | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)            | Returns the current cell matching the search criteria.                                                                                                           |
| [ignoreDiacritics(ignoreDiacritics)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#ignoreDiacritics(Boolean)) | [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder) | If `true`, configures the search to ignore diacritics while matching; otherwise the search matches diacritics.                                                   |
| [matchCase(matchCase)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#matchCase(Boolean))                      | [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder) | If `true`, configures the search to match the search text's case exactly, otherwise the search defaults to case-insensitive matching.                            |
| [matchEntireCell(matchEntireCell)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#matchEntireCell(Boolean))    | [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder) | If `true`, configures the search to match the entire contents of a cell; otherwise, the search defaults to partial matching.                                     |
| [matchFormulaText(matchFormulaText)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#matchFormulaText(Boolean)) | [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder) | If `true`, configures the search to return matches that appear within formula text; otherwise cells with formulas are considered based on their displayed value. |
| [replaceAllWith(replaceText)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#replaceAllWith(String))           | `Integer`                                                                                 | Replaces all matches with the specified text.                                                                                                                    |
| [replaceWith(replaceText)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#replaceWith(String))                 | `Integer`                                                                                 | Replaces the search text in the currently matched cell with the specified text and returns the number of occurrences replaced.                                   |
| [startFrom(startRange)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#startFrom(Range))                       | [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder) | Configures the search to start searching immediately after the specified cell range.                                                                             |
| [useRegularExpression(useRegEx)](https://developers.google.com/apps-script/reference/spreadsheet/text-finder#useRegularExpression(Boolean)) | [TextFinder](https://developers.google.com/apps-script/reference/spreadsheet/text-finder) | If `true`, configures the search to interpret the search string as a regular expression; otherwise the search interprets the search string as normal text.       |
