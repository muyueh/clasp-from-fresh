## [Selection](https://developers.google.com/apps-script/reference/spreadsheet/selection)

### Methods

|                                                                Method                                                                |                                       Return type                                       |                                                                                                                                                                                                                                                            Brief description                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getActiveRange()](https://developers.google.com/apps-script/reference/spreadsheet/selection#getActiveRange())                       | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)          | Returns the selected range in the active sheet, or `null` if there is no active range.                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [getActiveRangeList()](https://developers.google.com/apps-script/reference/spreadsheet/selection#getActiveRangeList())               | [RangeList](https://developers.google.com/apps-script/reference/spreadsheet/range-list) | Returns the list of active ranges in the active sheet or `null` if there are no active ranges.                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [getActiveSheet()](https://developers.google.com/apps-script/reference/spreadsheet/selection#getActiveSheet())                       | [Sheet](https://developers.google.com/apps-script/reference/spreadsheet/sheet)          | Returns the active sheet in the spreadsheet.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [getCurrentCell()](https://developers.google.com/apps-script/reference/spreadsheet/selection#getCurrentCell())                       | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)          | Returns the current (highlighted) cell that is selected in one of the active ranges or `null` if there is no current cell.                                                                                                                                                                                                                                                                                                                                                                                                               |
| [getNextDataRange(direction)](https://developers.google.com/apps-script/reference/spreadsheet/selection#getNextDataRange(Direction)) | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)          | Starting from the [current cell](https://developers.google.com/apps-script/reference/spreadsheet/selection#getCurrentCell()) and [active range](https://developers.google.com/apps-script/reference/spreadsheet/selection#getActiveRange()) and moving in the given direction, returns an adjusted range where the appropriate edge of the range has been shifted to cover the [next data cell](https://developers.google.com/apps-script/reference/spreadsheet/range#getNextDataCell(Direction)) while still covering the current cell. |
