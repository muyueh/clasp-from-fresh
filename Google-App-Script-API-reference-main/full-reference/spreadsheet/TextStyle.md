## [TextStyle](https://developers.google.com/apps-script/reference/spreadsheet/text-style)

### Methods

|                                                               Method                                                                |                                              Return type                                               |                              Brief description                               |
|-------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [copy()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#copy())                                         | [TextStyleBuilder](https://developers.google.com/apps-script/reference/spreadsheet/text-style-builder) | Creates a text style builder initialized with the values of this text style. |
| [getFontFamily()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#getFontFamily())                       | `String`                                                                                               | Gets the font family of the text.                                            |
| [getFontSize()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#getFontSize())                           | `Integer`                                                                                              | Gets the font size of the text in points.                                    |
| [getForegroundColorObject()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#getForegroundColorObject()) | [Color](https://developers.google.com/apps-script/reference/spreadsheet/color)                         | Gets the font color of the text.                                             |
| [isBold()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#isBold())                                     | `Boolean`                                                                                              | Gets whether or not the text is bold.                                        |
| [isItalic()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#isItalic())                                 | `Boolean`                                                                                              | Gets whether or not the cell is italic.                                      |
| [isStrikethrough()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#isStrikethrough())                   | `Boolean`                                                                                              | Gets whether or not the cell has strikethrough.                              |
| [isUnderline()](https://developers.google.com/apps-script/reference/spreadsheet/text-style#isUnderline())                           | `Boolean`                                                                                              | Gets whether or not the cell is underlined.                                  |
