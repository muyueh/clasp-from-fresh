## [SheetType](https://developers.google.com/apps-script/reference/spreadsheet/sheet-type)

### Properties

|   Property   |  Type  |                                                                       Description                                                                       |
|--------------|--------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| `GRID`       | `Enum` | A sheet containing a grid.                                                                                                                              |
| `OBJECT`     | `Enum` | A sheet containing a single embedded object such as an [EmbeddedChart](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart). |
| `DATASOURCE` | `Enum` | A sheet containing a [DataSource](https://developers.google.com/apps-script/reference/spreadsheet/data-source).                                         |
