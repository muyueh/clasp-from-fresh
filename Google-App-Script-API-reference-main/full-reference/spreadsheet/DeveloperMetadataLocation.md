## [DeveloperMetadataLocation](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location)

### Methods

|                                                               Method                                                               |                                                            Return type                                                            |                                                                                                     Brief description                                                                                                      |
|------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getColumn()](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location#getColumn())             | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)                                                    | Returns the [Range](https://developers.google.com/apps-script/reference/spreadsheet/range) for the column location of this metadata, or `null` if the location type is not `Developer``Metadata``Location``Type.COLUMN`.   |
| [getLocationType()](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location#getLocationType()) | [DeveloperMetadataLocationType](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location-type) | Gets the type of location.                                                                                                                                                                                                 |
| [getRow()](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location#getRow())                   | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range)                                                    | Returns the [Range](https://developers.google.com/apps-script/reference/spreadsheet/range) for the row location of this metadata, or `null` if the location type is not `Developer``Metadata``Location``Type.ROW`.         |
| [getSheet()](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location#getSheet())               | [Sheet](https://developers.google.com/apps-script/reference/spreadsheet/sheet)                                                    | Returns the [Sheet](https://developers.google.com/apps-script/reference/spreadsheet/sheet) location of this metadata, or `null` if the location type is not `Developer``Metadata``Location``Type.SHEET`.                   |
| [getSpreadsheet()](https://developers.google.com/apps-script/reference/spreadsheet/developer-metadata-location#getSpreadsheet())   | [Spreadsheet](https://developers.google.com/apps-script/reference/spreadsheet/spreadsheet)                                        | Returns the [Spreadsheet](https://developers.google.com/apps-script/reference/spreadsheet/spreadsheet) location of this metadata, or `null` if the location type is not `Developer``Metadata``Location``Type.SPREADSHEET`. |
