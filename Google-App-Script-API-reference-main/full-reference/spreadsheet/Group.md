## [Group](https://developers.google.com/apps-script/reference/spreadsheet/group)

### Methods

|                                                    Method                                                    |                                  Return type                                   |                                                                          Brief description                                                                           |
|--------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [collapse()](https://developers.google.com/apps-script/reference/spreadsheet/group#collapse())               | [Group](https://developers.google.com/apps-script/reference/spreadsheet/group) | Collapses this group.                                                                                                                                                |
| [expand()](https://developers.google.com/apps-script/reference/spreadsheet/group#expand())                   | [Group](https://developers.google.com/apps-script/reference/spreadsheet/group) | Expands this group.                                                                                                                                                  |
| [getControlIndex()](https://developers.google.com/apps-script/reference/spreadsheet/group#getControlIndex()) | `Integer`                                                                      | Returns the control toggle index of this group.                                                                                                                      |
| [getDepth()](https://developers.google.com/apps-script/reference/spreadsheet/group#getDepth())               | `Integer`                                                                      | Returns the depth of this group.                                                                                                                                     |
| [getRange()](https://developers.google.com/apps-script/reference/spreadsheet/group#getRange())               | [Range](https://developers.google.com/apps-script/reference/spreadsheet/range) | Returns the range over which this group exists.                                                                                                                      |
| [isCollapsed()](https://developers.google.com/apps-script/reference/spreadsheet/group#isCollapsed())         | `Boolean`                                                                      | Returns `true` if this group is collapsed.                                                                                                                           |
| [remove()](https://developers.google.com/apps-script/reference/spreadsheet/group#remove())                   | `void`                                                                         | Removes this group from the sheet, reducing the group depth of the [range](https://developers.google.com/apps-script/reference/spreadsheet/group#getRange()) by one. |
