# Gmail Service

Gmail

This Service lets you send email, compose drafts, manage labels, mark
messages and threads, and conduct a variety of other Gmail account management tasks.
See also [Mail Service](https://developers.google.com/apps-script/reference/mail), a simpler service
that only allows the sending of email.

## Classes

|                                             Name                                              |                    Brief description                    |
|-----------------------------------------------------------------------------------------------|---------------------------------------------------------|
| [GmailApp](https://developers.google.com/apps-script/reference/gmail/gmail-app)               | Provides access to Gmail threads, messages, and labels. |
| [GmailAttachment](https://developers.google.com/apps-script/reference/gmail/gmail-attachment) | An attachment from Gmail.                               |
| [GmailDraft](https://developers.google.com/apps-script/reference/gmail/gmail-draft)           | A user-created draft message in a user's Gmail account. |
| [GmailLabel](https://developers.google.com/apps-script/reference/gmail/gmail-label)           | A user-created label in a user's Gmail account.         |
| [GmailMessage](https://developers.google.com/apps-script/reference/gmail/gmail-message)       | A message in a user's Gmail account.                    |
| [GmailThread](https://developers.google.com/apps-script/reference/gmail/gmail-thread)         | A thread in a user's Gmail account.                     |
