## [GmailAttachment](https://developers.google.com/apps-script/reference/gmail/gmail-attachment)

### Methods

|                                                                      Method                                                                       |                           Return type                            |                                   Brief description                                   |
|---------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [copyBlob()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#copyBlob())                                               | [Blob](https://developers.google.com/apps-script/base/blob.html) | Returns a copy of this blob.                                                          |
| [getAs(contentType)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getAs(String))                                    | [Blob](https://developers.google.com/apps-script/base/blob.html) | Return the data inside this object as a blob converted to the specified content type. |
| [getBytes()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getBytes())                                               | `Byte[]`                                                         | Gets the data stored in this blob.                                                    |
| [getContentType()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getContentType())                                   | `String`                                                         | Gets the content type of the bytes in this blob.                                      |
| [getDataAsString()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getDataAsString())                                 | `String`                                                         | Gets the data of this blob as a String with UTF-8 encoding.                           |
| [getDataAsString(charset)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getDataAsString(String))                    | `String`                                                         | Gets the data of this blob as a string with the specified encoding.                   |
| [getHash()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getHash())                                                 | `String`                                                         | Gets the SHA1 content hash for this attachment.                                       |
| [getName()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getName())                                                 | `String`                                                         | Gets the name of this blob.                                                           |
| [getSize()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#getSize())                                                 | `Integer`                                                        | Gets the size of this attachment.                                                     |
| [isGoogleType()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#isGoogleType())                                       | `Boolean`                                                        | Returns whether this blob is a Google Workspace file (Sheets, Docs, etc.).            |
| [setBytes(data)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#setBytes(Byte))                                       | [Blob](https://developers.google.com/apps-script/base/blob.html) | Sets the data stored in this blob.                                                    |
| [setContentType(contentType)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#setContentType(String))                  | [Blob](https://developers.google.com/apps-script/base/blob.html) | Sets the content type of the bytes in this blob.                                      |
| [setContentTypeFromExtension()](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#setContentTypeFromExtension())         | [Blob](https://developers.google.com/apps-script/base/blob.html) | Sets the content type of the bytes in this blob based on the file extension.          |
| [setDataFromString(string)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#setDataFromString(String))                 | [Blob](https://developers.google.com/apps-script/base/blob.html) | Sets the data of this blob from a string with UTF-8 encoding.                         |
| [setDataFromString(string, charset)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#setDataFromString(String,String)) | [Blob](https://developers.google.com/apps-script/base/blob.html) | Sets the data of this blob from a string with the specified encoding.                 |
| [setName(name)](https://developers.google.com/apps-script/reference/gmail/gmail-attachment#setName(String))                                       | [Blob](https://developers.google.com/apps-script/base/blob.html) | Sets the name of this blob.                                                           |
