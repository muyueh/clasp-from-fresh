## [Selection](https://developers.google.com/apps-script/reference/slides/selection)

### Methods

|                                                       Method                                                        |                                            Return type                                            |                                                                                                                                                                                  Brief description                                                                                                                                                                                  |
|---------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getCurrentPage()](https://developers.google.com/apps-script/reference/slides/selection#getCurrentPage())           | [Page](https://developers.google.com/apps-script/reference/slides/page)                           | Returns the currently active [Page](https://developers.google.com/apps-script/reference/slides/page) or `null` if there is no active page.                                                                                                                                                                                                                                          |
| [getPageElementRange()](https://developers.google.com/apps-script/reference/slides/selection#getPageElementRange()) | [PageElementRange](https://developers.google.com/apps-script/reference/slides/page-element-range) | Returns the [PageElementRange](https://developers.google.com/apps-script/reference/slides/page-element-range) collection of [PageElement](https://developers.google.com/apps-script/reference/slides/page-element) instances that are selected or `null` if there are no [PageElement](https://developers.google.com/apps-script/reference/slides/page-element) instances selected. |
| [getPageRange()](https://developers.google.com/apps-script/reference/slides/selection#getPageRange())               | [PageRange](https://developers.google.com/apps-script/reference/slides/page-range)                | Returns the [PageRange](https://developers.google.com/apps-script/reference/slides/page-range) a collection of [Page](https://developers.google.com/apps-script/reference/slides/page) instances in the flimstrip that are selected or `null` if the selection is not of type [SelectionType.PAGE](https://developers.google.com/apps-script/reference/slides/selection-type#PAGE). |
| [getSelectionType()](https://developers.google.com/apps-script/reference/slides/selection#getSelectionType())       | [SelectionType](https://developers.google.com/apps-script/reference/slides/selection-type)        | Returns the [SelectionType](https://developers.google.com/apps-script/reference/slides/selection-type).                                                                                                                                                                                                                                                                             |
| [getTableCellRange()](https://developers.google.com/apps-script/reference/slides/selection#getTableCellRange())     | [TableCellRange](https://developers.google.com/apps-script/reference/slides/table-cell-range)     | Returns the [TableCellRange](https://developers.google.com/apps-script/reference/slides/table-cell-range) collection of [TableCell](https://developers.google.com/apps-script/reference/slides/table-cell) instances that are selected or `null` if there are no [TableCell](https://developers.google.com/apps-script/reference/slides/table-cell) instances selected.             |
| [getTextRange()](https://developers.google.com/apps-script/reference/slides/selection#getTextRange())               | [TextRange](https://developers.google.com/apps-script/reference/slides/text-range)                | Returns the [TextRange](https://developers.google.com/apps-script/reference/slides/text-range) that is selected or `null` if the selection is not of type [SelectionType.TEXT](https://developers.google.com/apps-script/reference/slides/selection-type#TEXT).                                                                                                                     |
