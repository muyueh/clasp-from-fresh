## [ListStyle](https://developers.google.com/apps-script/reference/slides/list-style)

### Methods

|                                                              Method                                                              |                                    Return type                                     |                                                                                                  Brief description                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [applyListPreset(listPreset)](https://developers.google.com/apps-script/reference/slides/list-style#applyListPreset(ListPreset)) | [ListStyle](https://developers.google.com/apps-script/reference/slides/list-style) | Applies the specified [ListPreset](https://developers.google.com/apps-script/reference/slides/list-preset) to all of the paragraphs that overlap with the text.                                                     |
| [getGlyph()](https://developers.google.com/apps-script/reference/slides/list-style#getGlyph())                                   | `String`                                                                           | Returns the rendered glyph for the text.                                                                                                                                                                            |
| [getList()](https://developers.google.com/apps-script/reference/slides/list-style#getList())                                     | [List](https://developers.google.com/apps-script/reference/slides/list)            | Returns the [List](https://developers.google.com/apps-script/reference/slides/list) the text is in, or `null` if none of the text is in a list, or part of the text is in a list, or the text is in multiple lists. |
| [getNestingLevel()](https://developers.google.com/apps-script/reference/slides/list-style#getNestingLevel())                     | `Integer`                                                                          | Returns the 0-based nesting level of the text.                                                                                                                                                                      |
| [isInList()](https://developers.google.com/apps-script/reference/slides/list-style#isInList())                                   | `Boolean`                                                                          | Returns `true` if the text is in exactly one list, `false` if none of the text is in a list, and `null` if only some of the text is in a list or if the text is in multiple lists.                                  |
| [removeFromList()](https://developers.google.com/apps-script/reference/slides/list-style#removeFromList())                       | [ListStyle](https://developers.google.com/apps-script/reference/slides/list-style) | Removes the paragraphs that overlap with the text from any lists.                                                                                                                                                   |
