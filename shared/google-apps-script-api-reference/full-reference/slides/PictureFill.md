## [PictureFill](https://developers.google.com/apps-script/reference/slides/picture-fill)

### Methods

|                                                   Method                                                    |                           Return type                            |                                   Brief description                                   |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [getAs(contentType)](https://developers.google.com/apps-script/reference/slides/picture-fill#getAs(String)) | [Blob](https://developers.google.com/apps-script/base/blob.html) | Return the data inside this object as a blob converted to the specified content type. |
| [getBlob()](https://developers.google.com/apps-script/reference/slides/picture-fill#getBlob())              | [Blob](https://developers.google.com/apps-script/base/blob.html) | Return the data inside this object as a blob.                                         |
| [getContentUrl()](https://developers.google.com/apps-script/reference/slides/picture-fill#getContentUrl())  | `String`                                                         | Gets a URL to the image.                                                              |
| [getSourceUrl()](https://developers.google.com/apps-script/reference/slides/picture-fill#getSourceUrl())    | `String`                                                         | Gets the image's source URL, if available.                                            |
