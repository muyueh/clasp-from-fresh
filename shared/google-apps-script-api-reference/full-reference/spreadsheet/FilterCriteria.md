## [FilterCriteria](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria)

### Methods

|                                                                   Method                                                                   |                                                   Return type                                                    |                                        Brief description                                         |
|--------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [copy()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#copy())                                           | [FilterCriteriaBuilder](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria-builder) | Copies this filter criteria and creates a criteria builder that you can apply to another filter. |
| [getCriteriaType()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#getCriteriaType())                     | [BooleanCriteria](https://developers.google.com/apps-script/reference/spreadsheet/boolean-criteria)              | Returns the criteria's boolean type, for example, `CELL_EMPTY`.                                  |
| [getCriteriaValues()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#getCriteriaValues())                 | `Object[]`                                                                                                       | Returns an array of arguments for boolean criteria.                                              |
| [getHiddenValues()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#getHiddenValues())                     | `String[]`                                                                                                       | Returns the values that the filter hides.                                                        |
| [getVisibleBackgroundColor()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#getVisibleBackgroundColor()) | [Color](https://developers.google.com/apps-script/reference/spreadsheet/color)                                   | Returns the background color used as filter criteria.                                            |
| [getVisibleForegroundColor()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#getVisibleForegroundColor()) | [Color](https://developers.google.com/apps-script/reference/spreadsheet/color)                                   | Returns the foreground color used as a filter criteria.                                          |
| [getVisibleValues()](https://developers.google.com/apps-script/reference/spreadsheet/filter-criteria#getVisibleValues())                   | `String[]`                                                                                                       | Returns the values that the pivot table filter shows.                                            |
