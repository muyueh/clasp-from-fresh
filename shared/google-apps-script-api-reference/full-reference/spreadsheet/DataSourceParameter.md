### Methods

|--------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getName()](https://developers.google.com/apps-script/reference/spreadsheet/data-source-parameter#getName())             | `String`                                                                                                              | Gets the parameter name.                                                                                                                                                                                                       |
| [getSourceCell()](https://developers.google.com/apps-script/reference/spreadsheet/data-source-parameter#getSourceCell()) | `String`                                                                                                              | Gets the source cell the parameter is valued based on, or `null` if the parameter type is not [DataSourceParameterType.CELL](https://developers.google.com/apps-script/reference/spreadsheet/data-source-parameter-type#CELL). |
| [getType()](https://developers.google.com/apps-script/reference/spreadsheet/data-source-parameter#getType())             | [DataSourceParameterType](https://developers.google.com/apps-script/reference/spreadsheet/data-source-parameter-type) | Gets the parameter type.                                                                                                                                                                                                       |
