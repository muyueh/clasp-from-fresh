## [EmbeddedChart](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart)

### Methods

|                                                                   Method                                                                    |                                                      Return type                                                      |                                                                        Brief description                                                                         |
|---------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [asDataSourceChart()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#asDataSourceChart())                   | [DataSourceChart](https://developers.google.com/apps-script/reference/spreadsheet/data-source-chart)                  | Casts to a data source chart instance if the chart is a data source chart, or `null` otherwise.                                                                  |
| [getAs(contentType)](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getAs(String))                          | [Blob](https://developers.google.com/apps-script/base/blob.html)                                                      | Return the data inside this object as a blob converted to the specified content type.                                                                            |
| [getBlob()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getBlob())                                       | [Blob](https://developers.google.com/apps-script/base/blob.html)                                                      | Return the data inside this object as a blob.                                                                                                                    |
| [getChartId()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getChartId())                                 | `Integer`                                                                                                             | Returns a stable identifier for the chart that is unique across the spreadsheet containing the chart or `null` if the chart is not in a spreadsheet.             |
| [getContainerInfo()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getContainerInfo())                     | [ContainerInfo](https://developers.google.com/apps-script/reference/spreadsheet/container-info)                       | Returns information about where the chart is positioned within a sheet.                                                                                          |
| [getHiddenDimensionStrategy()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getHiddenDimensionStrategy()) | [ChartHiddenDimensionStrategy](https://developers.google.com/apps-script/charts/chart-hidden-dimension-strategy.html) | Returns the strategy to use for handling hidden rows and columns.                                                                                                |
| [getMergeStrategy()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getMergeStrategy())                     | [ChartMergeStrategy](https://developers.google.com/apps-script/charts/chart-merge-strategy.html)                      | Returns the merge strategy used when more than one range exists.                                                                                                 |
| [getNumHeaders()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getNumHeaders())                           | `Integer`                                                                                                             | Returns the number of rows or columns the range that are treated as headers.                                                                                     |
| [getOptions()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getOptions())                                 | [ChartOptions](https://developers.google.com/apps-script/charts/chart-options.html)                                   | Returns the options for this chart, such as height, colors, and axes.                                                                                            |
| [getRanges()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getRanges())                                   | [Range[]](https://developers.google.com/apps-script/reference/spreadsheet/range)                                      | Returns the ranges that this chart uses as a data source.                                                                                                        |
| [getTransposeRowsAndColumns()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#getTransposeRowsAndColumns()) | `Boolean`                                                                                                             | If `true`, the rows and columns used to populate the chart are switched.                                                                                         |
| [modify()](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart#modify())                                         | [EmbeddedChartBuilder](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart-builder)        | Returns an [EmbeddedChartBuilder](https://developers.google.com/apps-script/reference/spreadsheet/embedded-chart-builder) that can be used to modify this chart. |
