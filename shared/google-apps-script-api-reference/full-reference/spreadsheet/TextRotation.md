## [TextRotation](https://developers.google.com/apps-script/reference/spreadsheet/text-rotation)

### Methods

|                                                   Method                                                   | Return type |                                 Brief description                                  |
|------------------------------------------------------------------------------------------------------------|-------------|------------------------------------------------------------------------------------|
| [getDegrees()](https://developers.google.com/apps-script/reference/spreadsheet/text-rotation#getDegrees()) | `Integer`   | Gets the angle between standard text orientation and the current text orientation. |
| [isVertical()](https://developers.google.com/apps-script/reference/spreadsheet/text-rotation#isVertical()) | `Boolean`   | Returns `true` if the text is stacked vertically; returns `false` otherwise.       |
