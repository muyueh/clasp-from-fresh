|              Property              |  Type  |                       Description                       |
|------------------------------------|--------|---------------------------------------------------------|

| `RUNNING`                          | `Enum` | The data execution has started and is running.          |
| `SUCCESS`                          | `Enum` | The data execution is completed and successful.         |
| `ERROR`                            | `Enum` | The data execution is completed and has errors.         |
