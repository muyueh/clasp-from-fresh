## [RichLink](https://developers.google.com/apps-script/reference/document/rich-link)

### Methods

|                                                          Method                                                           |                                            Return type                                             |                                                                              Brief description                                                                               |
|---------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [copy()](https://developers.google.com/apps-script/reference/document/rich-link#copy())                                   | [RichLink](https://developers.google.com/apps-script/reference/document/rich-link)                 | Returns a detached, deep copy of the current element.                                                                                                                        |
| [getAttributes()](https://developers.google.com/apps-script/reference/document/rich-link#getAttributes())                 | `Object`                                                                                           | Retrieves the element's attributes.                                                                                                                                          |
| [getMimeType()](https://developers.google.com/apps-script/reference/document/rich-link#getMimeType())                     | `String`                                                                                           | Returns the [MIME type](https://developers.google.com/drive/api/v3/mime-types) of the link, which is available when the item is a link to a Drive file and `null` otherwise. |
| [getNextSibling()](https://developers.google.com/apps-script/reference/document/rich-link#getNextSibling())               | [Element](https://developers.google.com/apps-script/reference/document/element)                    | Retrieves the element's next sibling element.                                                                                                                                |
| [getParent()](https://developers.google.com/apps-script/reference/document/rich-link#getParent())                         | [ContainerElement](https://developers.google.com/apps-script/reference/document/container-element) | Retrieves the element's parent element.                                                                                                                                      |
| [getPreviousSibling()](https://developers.google.com/apps-script/reference/document/rich-link#getPreviousSibling())       | [Element](https://developers.google.com/apps-script/reference/document/element)                    | Retrieves the element's previous sibling element.                                                                                                                            |
| [getTitle()](https://developers.google.com/apps-script/reference/document/rich-link#getTitle())                           | `String`                                                                                           | Returns the link's displayed title.                                                                                                                                          |
| [getType()](https://developers.google.com/apps-script/reference/document/rich-link#getType())                             | [ElementType](https://developers.google.com/apps-script/reference/document/element-type)           | Retrieves the element's [ElementType](https://developers.google.com/apps-script/reference/document/element-type).                                                            |
| [getUrl()](https://developers.google.com/apps-script/reference/document/rich-link#getUrl())                               | `String`                                                                                           | Returns the URL of the resource.                                                                                                                                             |
| [isAtDocumentEnd()](https://developers.google.com/apps-script/reference/document/rich-link#isAtDocumentEnd())             | `Boolean`                                                                                          | Determines whether the element is at the end of the [Document](https://developers.google.com/apps-script/reference/document/document).                                       |
| [merge()](https://developers.google.com/apps-script/reference/document/rich-link#merge())                                 | [RichLink](https://developers.google.com/apps-script/reference/document/rich-link)                 | Merges the element with the preceding sibling of the same type.                                                                                                              |
| [removeFromParent()](https://developers.google.com/apps-script/reference/document/rich-link#removeFromParent())           | [RichLink](https://developers.google.com/apps-script/reference/document/rich-link)                 | Removes the element from its parent.                                                                                                                                         |
| [setAttributes(attributes)](https://developers.google.com/apps-script/reference/document/rich-link#setAttributes(Object)) | [RichLink](https://developers.google.com/apps-script/reference/document/rich-link)                 | Sets the element's attributes.                                                                                                                                               |
