## [Position](https://developers.google.com/apps-script/reference/document/position)

### Methods

|                                                             Method                                                              |                                       Return type                                        |                                                                                                                                                                                                  Brief description                                                                                                                                                                                                   |
|---------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [getElement()](https://developers.google.com/apps-script/reference/document/position#getElement())                              | [Element](https://developers.google.com/apps-script/reference/document/element)          | Gets the element that contains this `Position`.                                                                                                                                                                                                                                                                                                                                                                      |
| [getOffset()](https://developers.google.com/apps-script/reference/document/position#getOffset())                                | `Integer`                                                                                | Gets this `Position`'s relative location within the element that contains it.                                                                                                                                                                                                                                                                                                                                        |
| [getSurroundingText()](https://developers.google.com/apps-script/reference/document/position#getSurroundingText())              | [Text](https://developers.google.com/apps-script/reference/document/text)                | Creates an artificial [Text](https://developers.google.com/apps-script/reference/document/text) element that represents the text and formatting of the [Paragraph](https://developers.google.com/apps-script/reference/document/paragraph) or [ListItem](https://developers.google.com/apps-script/reference/document/list-item) that contains the `Position`, either directly or through a chain of child elements. |
| [getSurroundingTextOffset()](https://developers.google.com/apps-script/reference/document/position#getSurroundingTextOffset())  | `Integer`                                                                                | Gets the offset of this `Position` within the [Text](https://developers.google.com/apps-script/reference/document/text) element returned by [getSurroundingText()](https://developers.google.com/apps-script/reference/document/position#getSurroundingText()).                                                                                                                                                      |
| [insertBookmark()](https://developers.google.com/apps-script/reference/document/position#insertBookmark())                      | [Bookmark](https://developers.google.com/apps-script/reference/document/bookmark)        | Creates and inserts a new [Bookmark](https://developers.google.com/apps-script/reference/document/bookmark) at this `Position`.                                                                                                                                                                                                                                                                                      |
| [insertInlineImage(image)](https://developers.google.com/apps-script/reference/document/position#insertInlineImage(BlobSource)) | [InlineImage](https://developers.google.com/apps-script/reference/document/inline-image) | Creates and inserts a new [InlineImage](https://developers.google.com/apps-script/reference/document/inline-image) at this `Position` from the specified image blob.                                                                                                                                                                                                                                                 |
| [insertText(text)](https://developers.google.com/apps-script/reference/document/position#insertText(String))                    | [Text](https://developers.google.com/apps-script/reference/document/text)                | Inserts the specified text at this `Position`.                                                                                                                                                                                                                                                                                                                                                                       |
