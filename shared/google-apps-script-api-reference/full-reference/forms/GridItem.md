| [getTitle()](https://developers.google.com/apps-script/reference/forms/grid-item#getTitle())                                   | `String`                                                                                | Gets the item's title (sometimes called header text, in the case of a [SectionHeaderItem](https://developers.google.com/apps-script/reference/forms/section-header-item)).                                                                                                                                                                                                    |

| [isRequired()](https://developers.google.com/apps-script/reference/forms/grid-item#isRequired())                               | `Boolean`                                                                               | Determines whether the respondent must answer the question.                                                                                                                                                                                                                                                                                                                   |

| [setRequired(enabled)](https://developers.google.com/apps-script/reference/forms/grid-item#setRequired(Boolean))               | [GridItem](https://developers.google.com/apps-script/reference/forms/grid-item)         | Sets whether the respondent must answer the question.                                                                                                                                                                                                                                                                                                                         |
